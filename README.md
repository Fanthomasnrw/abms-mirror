# ABMS Mirror

Official public documentation mirror of the Asphalt Burn Management System (ABMS).

> ⚠️ **This repository is NOT an open-source project.**

---

## Purpose

This repository provides the public operating standards, documentation, templates, and governance required for AI-assisted collaboration within the ABMS ecosystem.

It serves exclusively as a public knowledge mirror for AI systems and technical documentation.

---

## Scope

This repository may contain:

- Operating standards
- Project templates
- Documentation
- Governance rules
- Workflow definitions
- Technical handbooks

---

## Security Policy

No confidential information is stored in this repository.

The following content is intentionally excluded:

- Artwork
- Logos
- Product designs
- Source graphics
- Commercial assets
- Financial records
- Customer information
- Contracts
- Legal files
- Internal business documentation

The authoritative master copy of the ABMS always remains offline in the local company archive.

## Architecture

The ABMS follows a strict **One-Way Mirror** architecture.

```
Local Master Archive (Offline)
            │
            ▼
      ABMS Mirror (GitHub)
            │
            ▼
   AI Systems (ChatGPT, Gemini, Claude, ...)
```

The local master archive is the single authoritative source of truth.

The public mirror contains only documentation required for AI collaboration and never replaces the local archive.
